# Adira — AI Powered Teacher's Assistant

Adira is a WhatsApp-based AI assistant for Indian school teachers. 
It generates CBSE-aligned educational documents instantly.

## What Adira Does
- 📝 Lesson Plans — CBSE format, Bloom's taxonomy
- 📋 Worksheets — Chapter-wise with answer keys  
- 📄 Question Papers — Official CBSE pattern

## How It Works
Teacher sends a WhatsApp message like:
"worksheet class 7 science chapter 3"

Adira responds with a professional PDF in 30 seconds.

## Technology
- AWS Lambda (Node.js)
- Claude AI (Anthropic)
- WhatsApp Business API
- NCERT Knowledge Base (14,805 chunks)

## Target Users
CBSE school teachers across India (Classes 3-12)
